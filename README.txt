# Tutorial - Rodando o WebScrapping

1- instale o python pelo link https://python.org. 
Instale a versão mais recente. 

2- Execute o arquivo "install.bat" (clique duas vezes). Ele
vai instalar todas as dependências para que o programa funcione

3- Execute o arquivo "main.py" (clique duas vezes). Quando quiser finalizar 
a execução, pressione ESC
